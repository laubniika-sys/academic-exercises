# 💳 Bank of Sasha

Projeto desenvolvido para a disciplina de Análise e Desenvolvimento de Sistemas na Universidade Cruzeiro do Sul (UNICSUL).

## 📌 Descrição

O **Bank of Sasha** é uma aplicação web simples que simula operações bancárias básicas, permitindo ao usuário interagir com um sistema de forma prática e intuitiva.

O projeto foi originalmente baseado em lógica desenvolvida em Python e posteriormente adaptado para uma aplicação web utilizando HTML, CSS e JavaScript.

---

## ⚙️ Funcionalidades

* 💰 Visualização de saldo
* ➕ Depósito de valores
* 🌍 Depósito internacional com conversão de moedas:

  * Euro
  * Real
  * Yen
  * Pound
* ➖ Saque de valores
* 💾 Persistência de dados utilizando **LocalStorage**

---

## 🖥️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

---

## 🎨 Interface

A interface foi inspirada em aplicações de bancos digitais modernos, com foco em:

* Design limpo
* Botões intuitivos
* Feedback visual ao usuário
* Organização em múltiplas páginas
